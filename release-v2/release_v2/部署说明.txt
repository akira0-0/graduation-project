这是最终的自动化运行包。为了保证服务器上不报错，请务必执行以下 4步 操作：

1. 新增调度脚本 请将 auto_hot_runner.py 放在项目根目录下（和 xhs-crawler/main.py 同级）。

作用：每天 09:00 全自动抓热搜 -> 更新配置 -> 依次调度三大爬虫。

2. 替换微博爬虫入口 (关键修复) 请用附件中的 weibo_main_fix.py，重命名为 main.py，然后覆盖掉项目中的： xhs-crawler/weibo_crawler/weibo_crawler/main.py

原因：修复了 Linux 环境下的 UTF-8 编码报错问题。

3. 替换 Web 爬虫入口 (关键修复) 请用附件中的 web_spider_fix.py，重命名为 spider.py，然后覆盖掉项目中的： xhs-crawler/web_crawler/spider.py

原因：同样修复了中文编码问题，防止抓取网易新闻时乱码崩溃。

4. 服务器依赖与启动 请确保安装了以下库（如果已装过可跳过）：

Bash

pip install schedule playwright
playwright install chromium  

5. 正式启动 直接后台挂起这个调度器即可：

Bash

nohup python auto_hot_runner.py > scheduler.log 2>&1 &
